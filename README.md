# Short Stays

A React application for managing apartment listings.

## Features

- View, create, edit, and delete apartment listings
- Detail view for individual apartments
- Client-side routing with React Router

## Tech Stack

- React 19
- Vite
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
